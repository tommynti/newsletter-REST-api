# newsletter-REST-api

REST API made with Node and MongoDB.

## Routes 

```
POST  /api/articles
GET    /api/articles/id/:arid/:payload?
GET    /api/articles/:contents?/:category?
PATCH  /api/articles/id/:arid
DELETE /api/articles/id/:arid
POST   /api/categories
GET    /api/categories
DELETE /api/categories/:cid
```
